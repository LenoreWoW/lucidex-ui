# Installation & Setup

This guide will walk you through installing and setting up Lucidex UI Explorer in your project, regardless of which framework you're using.

## Prerequisites

Before installing Lucidex UI, make sure you have:

- **Node.js** 16.x or later (for React/Next.js projects)
- **.NET 6.0** or later (for Blazor projects)
- A modern browser that supports ES6+ features

## Framework-Specific Installation

### React

For React applications, install the core React package:

  language="jsx"
  code={`import React from 'react';
import { PolarisProvider } from '@polaris-ui/react';
import '@polaris-ui/react/styles.css';

function App() {
return (
<PolarisProvider theme="light">
<div className="App">
{/* Your app content */}

}

/>

<CodeExample

      <body>

/>

### Blazor Server & WebAssembly

For Blazor applications:

<InstallationSteps
  title="Blazor Installation"
  steps={[
    'dotnet add package PolarisUI.Blazor',
    'Add CSS reference to index.html or _Host.cshtml',
    'Register services in Program.cs',
    'Add using statement to _Imports.razor',
  ]}
/>

<CodeExample
  title="Program.cs"
  language="csharp"
  code={`using PolarisUI.Blazor;

var builder = WebAssemblyHostBuilder.CreateDefault(args);
builder.RootComponents.Add<App>("#app");

// Register Lucidex UI services
builder.Services.AddPolarisUI();

await builder.Build().RunAsync();`}
/>

<CodeExample
  title="index.html"
  language="html"
  code={`<!DOCTYPE html>

</html>`}
/>

## Troubleshooting

### Common Issues

**Styles not loading**

- Ensure you've imported the CSS file correctly
- Check that the file path is correct for your bundler
- Verify there are no conflicting CSS rules

**Components not rendering**

- Make sure you've wrapped your app with `PolarisProvider`
- Check for JavaScript errors in the browser console
- Verify all required dependencies are installed

**TypeScript errors**

- Install `@polaris-ui/types` package
- Add types to your `tsconfig.json`
- Restart your TypeScript language server

### Getting Help

If you're still having issues:
