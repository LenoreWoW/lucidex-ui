# Getting Started with Lucidex UI Explorer

Welcome to the Lucidex UI Explorer! This comprehensive design system helps you build beautiful, consistent user interfaces across multiple frameworks including React, Next.js, Blazor, HTML, and TypeScript.

## What is Lucidex UI Explorer?

The Lucidex UI Explorer is a modern, framework-agnostic design system that provides:

- **Consistent Design Tokens**: Colors, typography, spacing, and more
- **Multi-Framework Support**: Components for React, Next.js, Blazor, HTML, and TypeScript
- **Interactive Examples**: Live code examples and component previews
- **Theming System**: Light and dark mode support with customizable themes
- **Accessibility First**: Built with WCAG compliance in mind
